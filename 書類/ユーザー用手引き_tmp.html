はじめに

PRM Toolとは

PRM Tool（Partner Relationship Management Tool）は、パートナー企業との関係管理、プロジェクト管理、手数料管理、請求書管理を統合的に行うWebアプリケーションです。

対象ユーザー

  • SYSTEM: システム管理者（初期セットアップ専用）
  • ADMIN: 管理者（全機能へのアクセス）
  • ACCOUNTING: 会計担当（手数料・請求書管理）
  • REP: 営業担当（プロジェクトの閲覧・編集）

システム要件

  • ブラウザ: Google Chrome、Firefox、Safari、Edge（最新版推奨）
  • 画面解像度: 1280×720 以上推奨
  • インターネット接続: 必須

ログイン

ログイン手順

  1. ブラウザでPRM Toolのログイン画面を開く
  2. ログインIDとパスワードを入力
  3. 「ログイン」ボタンをクリック

初回ログイン時の注意

  • 初回ログイン時は、管理者から提供された仮パスワードを使用してください
  • ログイン後、パスワードの変更を推奨します

ログインできない場合

  • ログインIDとパスワードが正しいか確認
  • Caps Lockがオンになっていないか確認
  • 管理者に連絡して、アカウントが有効化されているか確認

ダッシュボード

概要

ダッシュボードでは、以下の情報が一目で確認できます：

  • パートナー数: 登録されているパートナー企業の総数
  • プロジェクト数: 進行中のプロジェクト数
  • 手数料ルール数: 登録されている手数料ルールの総数
  • 請求書数: 発行された請求書の総数

アカウント管理

対象ロール: SYSTEM、ADMIN

ユーザー一覧表示

  1. 左メニューから「アカウント管理」をクリック
  2. 登録されているユーザーの一覧が表示されます

ユーザー作成

  1. 「新規ユーザー作成」ボタンをクリック
  2. 必須項目を入力：
    • ユーザー名: 氏名
    • ログインID: ログイン時に使用するID
    • パスワード: 8文字以上
    • メールアドレス: 連絡先メール
    • 電話番号: 連絡先電話番号
    • ロール: ADMIN / ACCOUNTING / REP から選択
  3. 「作成」ボタンをクリック

ユーザー編集

  1. ユーザー一覧から編集したいユーザーを選択
  2. 「編集」ボタンをクリック
  3. 必要な項目を変更
  4. 「保存」ボタンをクリック

ユーザー削除

  1. ユーザー一覧から削除したいユーザーを選択
  2. 「削除」ボタンをクリック
  3. 確認ダイアログで「削除」をクリック

注意: システム保護されたユーザー（SYSTEM権限）は削除できません。

パートナー管理

対象ロール: ADMIN、ACCOUNTING、REP

パートナー一覧表示

  1. 左メニューから「パートナー管理」をクリック
  2. 登録されているパートナー企業の一覧が表示されます

パートナー作成

  1. 「新規パートナー作成」ボタンをクリック
  2. 必須項目を入力：
    • 企業名: パートナー企業の名称
    • 業界: 業種
    • 電話番号: 代表電話（例: 03-1234-5678）
    • 郵便番号: 7桁の数字（例: 1234567）
    • 住所: 所在地
    • メールアドレス: 企業代表メール
  3. 担当者情報を追加：
    • 担当者名: 担当者の氏名
    • 部署: 所属部署
    • 役職: 役職
    • 電話番号: 直通電話
    • メールアドレス: 担当者メール
  4. 「作成」ボタンをクリック

郵便番号自動検索機能

郵便番号を入力すると、住所が自動的に入力されます：

  1. 郵便番号フィールドに7桁の数字を入力
  2. 自動的に住所が検索されて入力されます
  3. 必要に応じて住所を編集できます

パートナー編集

  1. パートナー一覧から編集したい企業を選択
  2. 「編集」ボタンをクリック
  3. 必要な項目を変更
  4. 「保存」ボタンをクリック

パートナー削除

  1. パートナー一覧から削除したい企業を選択
  2. 「削除」ボタンをクリック
  3. 確認ダイアログで「削除」をクリック

注意: プロジェクトが紐付いているパートナーは削除できません。

プロジェクト管理

対象ロール: ADMIN、ACCOUNTING、REP

プロジェクト一覧表示

  1. 左メニューから「プロジェクト管理」をクリック
  2. 登録されているプロジェクトの一覧が表示されます

プロジェクト作成

  1. 「新規プロジェクト作成」ボタンをクリック
  2. 必須項目を入力：
    • プロジェクト名: プロジェクトの名称
    • パートナー: 関連するパートナー企業を選択
    • 開始日: プロジェクト開始日
    • 終了日: プロジェクト終了予定日
    • ステータス: 新規 / 進行中 / 完了
    • 説明: プロジェクトの概要
  3. 担当者を追加：
    • 複数の担当者を選択可能
  4. 「作成」ボタンをクリック

スプレッドシート機能

プロジェクトには、Excel風のスプレッドシート機能が備わっています：

  1. プロジェクト詳細画面を開く
  2. 「スプレッドシート」タブをクリック
  3. セルをクリックして編集
  4. 行や列を追加・削除
  5. 「保存」ボタンで変更を保存

スプレッドシート機能の特徴:

  • セルの結合
  • 書式設定（太字、斜体、文字色、背景色）
  • 数式計算（SUM、AVERAGEなど）
  • 自動保存機能

プロジェクト編集

  1. プロジェクト一覧から編集したいプロジェクトを選択
  2. 「編集」ボタンをクリック
  3. 必要な項目を変更
  4. 「保存」ボタンをクリック

プロジェクト削除

  1. プロジェクト一覧から削除したいプロジェクトを選択
  2. 「削除」ボタンをクリック
  3. 確認ダイアログで「削除」をクリック

注意: 手数料ルールが紐付いているプロジェクトは削除できません。

手数料ルール管理

対象ロール: ADMIN、ACCOUNTING

手数料ルール一覧表示

  1. 左メニューから「手数料ルール」をクリック
  2. 登録されている手数料ルールの一覧が表示されます

手数料ルール作成

  1. 「新規ルール作成」ボタンをクリック
  2. 必須項目を入力：
    • ルール名: 手数料ルールの名称
    • プロジェクト: 関連するプロジェクトを選択
    • 手数料タイプ: 固定金額 / 割合
    • 金額または割合:
      • 固定金額の場合: 金額を入力（例: 100000）
      • 割合の場合: パーセンテージを入力（例: 10）
    • 説明: ルールの詳細説明
  3. 「作成」ボタンをクリック

手数料タイプの説明

固定金額

数量に関係なく固定の金額を適用します。

例:

  • 手数料: 100,000円
  • 商品数量: 10個
  • 合計手数料: 100,000円

割合

商品金額に対する割合を適用します。

例:

  • 手数料: 10%
  • 商品金額: 1,000,000円
  • 合計手数料: 100,000円

手数料ルール確定

  1. 手数料ルール一覧から確定したいルールを選択
  2. 「確定」ボタンをクリック
  3. 確認ダイアログで「確定」をクリック

注意: 確定後は編集・削除ができなくなります。

手数料ルール削除

  1. 手数料ルール一覧から削除したいルール（下書きステータス）を選択
  2. 「削除」ボタンをクリック
  3. 確認ダイアログで「削除」をクリック

請求書管理

対象ロール: ADMIN、ACCOUNTING

請求書一覧表示

  1. 左メニューから「請求書管理」をクリック
  2. 発行された請求書の一覧が表示されます

請求書作成

  1. 「新規請求書作成」ボタンをクリック
  2. 必須項目を入力：
    • パートナー: 請求先パートナーを選択
    • 請求日: 請求書の発行日
    • 支払期限: 支払い期限
    • 手数料ルール: 適用する手数料ルールを選択
    • 商品名: 商品またはサービス名
    • 数量: 商品の数量
    • 単価: 商品の単価
  3. 「作成」ボタンをクリック

自動計算機能

請求書では、以下が自動的に計算されます：

  • 小計: 数量 × 単価
  • 手数料: 小計 × 手数料ルール
  • 消費税: （小計 + 手数料）× 10%
  • 合計金額: 小計 + 手数料 + 消費税

請求書ダウンロード（PDF）

  1. 請求書一覧から該当する請求書を選択
  2. 「PDFダウンロード」ボタンをクリック
  3. PDF形式で請求書がダウンロードされます

請求書編集

  1. 請求書一覧から編集したい請求書を選択
  2. 「編集」ボタンをクリック
  3. 必要な項目を変更
  4. 「保存」ボタンをクリック

請求書削除

  1. 請求書一覧から削除したい請求書を選択
  2. 「削除」ボタンをクリック
  3. 確認ダイアログで「削除」をクリック

コンテンツ管理

対象ロール: ADMIN、ACCOUNTING、REP

概要

コンテンツ管理機能では、ファイルやフォルダーを一元管理し、パートナー企業と安全に共有できます。

フォルダー作成

  1. 左メニューから「コンテンツ管理」をクリック
  2. 「フォルダ作成」ボタンをクリック
  3. 必須項目を入力：
    • フォルダ名: フォルダーの名称
    • 説明: フォルダーの説明（任意）
  4. 親フォルダーを選択している場合、サブフォルダーとして作成されます
  5. 「作成」ボタンをクリック

フォルダー階層

フォルダーは階層構造で管理できます：

📁 製品カタログ
  📂 2024年版
    📂 国内向け
    📂 海外向け
  📂 2025年版
📁 契約書類
  📂 標準契約書
  📂 秘密保持契約書

ファイルアップロード

  1. フォルダーを選択
  2. 「ファイルアップロード」ボタンをクリック
  3. アップロードするファイルを選択（複数選択可能）
  4. 「アップロード」ボタンをクリック

対応ファイル形式:

  • ドキュメント: PDF、Word、Excel、PowerPoint
  • 画像: JPG、PNG、GIF
  • その他: ZIP、CSV、TXT

ファイルサイズ制限: 1ファイルあたり最大50MB

ファイルダウンロード

  1. ダウンロードしたいファイルを選択
  2. 「ダウンロード」ボタンをクリック
  3. ファイルがダウンロードされます

お気に入り機能

よく使うフォルダーをお気に入りに登録できます：

  1. フォルダーを選択
  2. 右上の「☆ お気に入りに追加」ボタンをクリック
  3. 左サイドバーの「⭐ お気に入り」セクションに追加されます

お気に入り機能の特徴:

  • 最大10個まで登録可能
  • ワンクリックでアクセス
  • 個人ごとに管理

お気に入り解除

  1. お気に入り登録済みのフォルダーを選択
  2. 右上の「⭐ お気に入り解除」ボタンをクリック

パンくずリスト

フォルダー階層がパンくずリストで表示されます：

浅い階層の場合:

ホーム › 製品カタログ › 2024年版

深い階層の場合:

ホーム › ... › 東京 › 23区 › 港区

パンくずリストをクリックすると、その階層に移動できます。

サブフォルダー表示

フォルダーを選択すると、以下が表示されます：

  1. サブフォルダー: 1階層下のフォルダー
  2. ファイル: そのフォルダー内のファイル

コンテンツ共有

対象ロール: ADMIN、ACCOUNTING

コンテンツをパートナー企業と共有できます：

  1. 「コンテンツ共有」メニューをクリック
  2. 「新規共有作成」ボタンをクリック
  3. 必須項目を入力：
    • ファイル: 共有するファイルを選択
    • 共有先: 特定パートナー / 全パートナー
    • 有効期限: 共有の有効期限（任意）
    • ダウンロード制限: 最大ダウンロード回数（任意）
    • メッセージ: 共有相手へのメッセージ（任意）
  4. 「共有」ボタンをクリック

共有ステータス

  • 有効: 共有が有効
  • 期限切れ: 有効期限を過ぎた
  • 制限到達: ダウンロード回数制限に到達
  • 無効化: 手動で無効化された

共有の無効化

  1. 共有一覧から無効化したい共有を選択
  2. 「無効化」ボタンをクリック
  3. 確認ダイアログで「無効化」をクリック

トラブルシューティング

よくある質問

Q1: ログインできません

A: 以下を確認してください：

  • ログインIDとパスワードが正しいか
  • Caps Lockがオンになっていないか
  • ブラウザのCookieが有効になっているか

それでも解決しない場合は、管理者に連絡してください。

Q2: ファイルがアップロードできません

A: 以下を確認してください：

  • ファイルサイズが50MB以下か
  • 対応ファイル形式か
  • インターネット接続が安定しているか

Q3: パンくずリストが表示されません

A: ブラウザをリロード（Ctrl+Shift+R / Cmd+Shift+R）してください。

Q4: 請求書がPDF出力できません

A: ブラウザのポップアップブロックを無効にしてください。

Q5: お気に入りが登録できません

A: お気に入りは最大10個までです。不要なお気に入りを削除してから再度登録してください。

エラーメッセージ

「権限がありません」

このエラーは、現在のユーザーロールでは実行できない操作を行おうとした場合に表示されます。

対処法: 管理者に必要な権限を依頼してください。

「ファイルの取得に失敗しました」

このエラーは、ファイルの読み込み中にエラーが発生した場合に表示されます。

対処法:

  1. ブラウザをリロード
  2. インターネット接続を確認
  3. それでも解決しない場合は、管理者に連絡

「お気に入りフォルダーは最大10個までです」

このエラーは、お気に入りの上限に達している場合に表示されます。

対処法: 不要なお気に入りを削除してから再度登録してください。

ブラウザ別の推奨設定

  • JavaScriptを有効化

  • JavaScriptを有効化
  • Cookieを有効化
  • プライベートブラウズモードを無効化
